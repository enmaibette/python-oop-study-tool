## Description

In this challenge, you will extend the `Dog` class by adding a `bark()` method. Methods are functions defined inside a class that describe the behaviors of objects created from that class.

## Requirements

- Extend the `Dog` class from the previous challenge
- Add a `bark()` method to the `Dog` class
- Print `Woof! My name is {name}.` in `bark()`, where `{name}` is the dog's name
## Example

```python
dog = Dog("Rex")
dog.bark()  # Output: Woof! My name is Rex.
```
